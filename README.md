## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/timstoldt/portfolio.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd portfolio
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Run the development server:**
   ```
   bash
   npm start
   ```
